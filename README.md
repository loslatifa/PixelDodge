# PixelDodge

> 一款基于 **Swift** 和 **SpriteKit** 开发的 macOS 像素风实时闪避游戏。

---

## 项目简介

PixelDodge 是一个使用 **Swift** 和 **SpriteKit** 开发的 macOS 2D 像素风街机闪避游戏项目。

玩家需要控制角色在场景中移动，躲避不断出现的危险方块，收集金币并尽可能生存更长时间。随着游戏进行，节奏会逐步加快，玩家需要在更高压力下完成闪避与路径判断。

这个项目不仅是一个可运行的小游戏，也是一项围绕以下内容展开的个人实践：

- 实时游戏循环设计
- 角色移动与碰撞检测
- 难度递增与关卡节奏控制
- 本地存档与解锁机制
- 基于 SpriteKit 的 macOS 游戏开发流程

## 当前功能

目前项目已经实现以下核心内容：

- 角色移动与基础闪避玩法
- 分数累计与金币收集
- 随时间递增的无限关卡机制
- 基础生命值与失败判定
- 简易存档与关卡解锁
- 全屏自适应显示
- 纯本地运行，无需联网

## 开发状态

目前项目处于持续迭代阶段，已经具备基础可玩性，但整体玩法深度和内容丰富度仍在完善中。

正在推进的优化方向包括：

## 操作方式

- **方向键 / WASD**：控制角色移动
- **ESC**：暂停游戏
- **空格**：继续游戏
- **拾取金币**：增加分数
- **碰到敌人**：损失生命值，生命值耗尽后游戏结束

## 技术栈

- **语言**：Swift
- **框架**：SpriteKit
- **平台**：macOS

## 运行方式

### 本地运行

1. 克隆仓库
2. 使用 Xcode 打开项目中的 `.xcodeproj`
3. 按下 `Command + R` 运行项目

### 分发给其他人

- 可通过 `Xcode > Product > Archive > Export App` 导出应用
- 也可以手动打包为 `.dmg` 进行分享

如果 macOS 出现安全拦截，可通过“右键打开”后允许应用运行。
